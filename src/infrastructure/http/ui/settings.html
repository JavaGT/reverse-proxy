Server settings

Values here are stored in SQLite (meta.server_settings) and merged over .env when the process starts or after you save. SQLITE_DB_PATH (and usually TLS_CERT_DIR for first boot) stay in .env only.

DNS console links (registrar “manage DNS” URLs on the Domains page) resolve in this order: per-apex override → default set on Domains → this setting (dnsConsoleDefaultProvider) → .env DNS_CONSOLE_DEFAULT_PROVIDER.

Changing listener port, TLS path, session secret, trust proxy, or rate limits may require a full process restart to apply everywhere.
