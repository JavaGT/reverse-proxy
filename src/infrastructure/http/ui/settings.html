Server settings

Values here are stored in SQLite (meta.server_settings) and merged with built-in defaults when the process starts or after you save. The database file is always ./reverse-proxy.db in the server’s working directory.

DNS console links (registrar “manage DNS” URLs on the Domains page) resolve in this order: per-apex override → default set on Domains → this setting (dnsConsoleDefaultProvider).

Changing listener port, TLS path, session secret, trust proxy, or rate limits may require a full process restart to apply everywhere.
